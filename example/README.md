# Shuttlers Live Chat - Example App

Demo application for testing the `shuttlers_live_chat` package.

## Running the Example

- fvm flutter run


- Testing Typing Indicators and Presence
- Connection State**: WiFi icon should turn green when connected
- Badge shows number of connected users
-  Send and receive messages in real-time
- Disable/enable network to test auto-reconnect

## Configuration

- Get a valid JWT token from your backend or JWT builder
- Use any trip identifier (default: `trip_001`)
- Enter a unique name for each instance
- Default points to QA environment
